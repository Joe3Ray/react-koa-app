# react-koa-app

> 基于 React 和 Koa2 的项目

## 初始化

clone 项目模板到本地，删除目录中的`.git`目录

```sh
git clone https://github.com/Joe3Ray/react-koa-app.git my-project
cd my-project && rm -rf .git
```

## 环境依赖

- node 8+ LTS
- npm 5+

## 本地开发

```sh
npm i
npm start
```

## 常用命令

JS 代码风格检查

```sh
npm run lint
```

单元测试

```sh
npm test
```
